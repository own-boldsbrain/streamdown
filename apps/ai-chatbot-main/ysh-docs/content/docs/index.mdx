# Bem-vindo à documentação AP2

O Auto Portal Fotovoltaico (AP2) é uma plataforma completa para gestão do ciclo de vida de sistemas fotovoltaicos, utilizando agentes especializados, Model Context Protocol (MCP) e mensageria NATS.

## Sobre o AP2

O AP2 é construído com uma arquitetura de agentes especializados que atuam em três fases principais:

1. **PRE (Planejamento)** - Desde a captação de leads até a proposta comercial
2. **ONGOING (Execução)** - Da engenharia à instalação
3. **POST (Operação)** - Monitoramento, expansão e outros serviços pós-instalação

Cada agente utiliza ferramentas específicas via MCP (Model Context Protocol) e se comunica com outros componentes através do NATS, um sistema de mensageria de alto desempenho.

A interface do usuário apresenta componentes visuais especializados (Artifacts) para cada etapa do processo, oferecendo uma experiência rica e informativa.
